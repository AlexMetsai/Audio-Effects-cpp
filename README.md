# Audio-Effects-cpp

Implementations of audio effects on the c++ language. 

This code relies on the header file provided by https://github.com/adamstark/AudioFile for wav audio I/O.

Compile with:
******************************
g++ reverb.cpp -std=c++11 -fpermissive
******************************

Example run:
******************************
./a.out -i input.wav
******************************
